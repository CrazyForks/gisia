# Release doc of 0.1.1

## What's new

1. New feature: Project Board
1. bug fixes

## Actons

1. change the gisia version to `image: gisia/gisia:0.1.2` in your `docker-compose.yml`
1. make a new dir `shared` in your project folder
1. change the folder owner by `sudo chown -R 1010:1010 shared`
1. update the `docker-compose.yml` with an additional mount `./shared:/rails/shared`

```yaml
+      - ./shared:/rails/shared
       - ./builds:/rails/builds
```
