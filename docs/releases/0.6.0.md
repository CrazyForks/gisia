# Release doc of 0.6.0

## What's new

1. Add REST API for PAT, branches, tags, pipelines, jobs, members, and users
2. Add activity tracking with timeline support for issues, epics, and merge requests
3. Add diff versions and resolve feature for diff comments
4. Add squash enabled support for merge requests
5. Fix note policy, project menus, and notification errors


## Actions

1. Change the gisia version to `image: gisia/gisia:0.6.0` in your `docker-compose.yml`
